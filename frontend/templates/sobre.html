Sobre o Projeto

O PlayMood é um sistema inteligente que recomenda playlists com base no seu humor. Ele utiliza análise de sentimentos para entender como você está se sentindo e encontra músicas que combinam — ou contrastam — com o seu estado emocional atual.

As recomendações são buscadas em tempo real no Spotify e cada sugestão é apresentada com um toque de humor e personalização.

Este projeto foi desenvolvido usando Python, FastAPI e a API do Spotify. O objetivo é ser divertido, útil e mostrar o poder de integração entre tecnologia e emoções.

Se quiser contribuir ou acompanhar o desenvolvimento, acesse o repositório no GitHub.
